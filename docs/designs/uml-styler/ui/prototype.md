## 1. 布局概述

│                                                                              │
│  ┌─────────────────────────────┐  ┌─────────────────────────────────────┐   │
│  │                             │  │                                     │   │
│  │       代码编辑器            │  │          预览区域                   │   │
│  │       (Monaco)             │  │                                     │   │
│  │                             │  │    ┌─────────────────────┐         │   │
│  │  line 1: code...           │  │    │                     │         │   │
│  │  line 2: code...           │  │    │    [渲染的图表]     │         │   │

│  │                             │  │                                     │   │

│  ├─────────────────────────────┤  ├─────────────────────────────────────┤   │

**布局比例：**
- 编辑器区域：40%
- 预览区域：60%
- 最小编辑器宽度：400px
- 最小预览宽度：500px

- 滑动切换 Tab（可选）

## 2. 组件设计

### 2.1 编辑器组件

```
┌─────────────────────────────────────────┐
│  [Mermaid ▼]  [模板 ▼]  [格式化] [清空] │ ← 工具栏
├─────────────────────────────────────────┤
│  1 │ sequenceDiagram                    │
│  2 │     participant A as Alice        │
│  3 │     participant B as Bob          │
│  4 │     A->>B: Hello!                 │
│  5 │     B-->>A: Hi!        [错误波浪线]│
│  6 │                                    │
│  └─────────────────────────────────────┘
│                                         │
│  Ln 5, Col 12    │    Mermaid Syntax    │ ← 状态栏
└─────────────────────────────────────────┘
```

**功能要点：**
- 工具栏：引擎切换、模板选择、格式化、清空
- 行号显示
- 语法高亮
- 错误波浪线标记
- 状态栏：行列位置、语法类型

### 2.2 预览组件

```
┌─────────────────────────────────────────────────────┐
│  [🔍+ 100% 🔍-] [适应] [全屏]                       │ ← 缩放控制
├─────────────────────────────────────────────────────┤
│                                                     │
│        ┌─────────────────────────────────┐        │
│        │                                 │        │
│        │      [SVG 图表内容]            │        │
│        │                                 │        │
│        │                                 │        │
│        └─────────────────────────────────┘        │
│                                                     │
│        (可拖拽、可缩放)                            │
│                                                     │
├─────────────────────────────────────────────────────┤
│  错误面板 (仅错误时显示):                           │
│  ┌───────────────────────────────────────────────┐ │
│  │ ⚠️ Line 5, Col 12                            │ │
│  │    Expecting '->>', got '-->'                │ │
│  │    [跳转到错误位置]                           │ │
│  └───────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────┘
```

**交互方式：**
- 鼠标滚轮：缩放
- 拖拽：平移
- 双击：适应窗口
- 错误面板：点击跳转到编辑器对应行

## 3. 响应式断点

| 断点名称 | 宽度范围 | 布局方式 | 关键调整 |
|----------|---------|----------|----------|
| Desktop | ≥1024px | 左右分屏 | 完整功能 |
| Tablet | 768-1023px | 上下分屏 | 折叠主题微调面板 |
| Mobile | <768px | Tab 导航 | 单屏单功能 |

## 4. 组件尺寸规范

### 4.1 最小尺寸

| 组件 | 最小宽度 | 最小高度 |
|------|---------|---------|
| 编辑器区域 | 400px | 300px |
| 预览区域 | 500px | 400px |
| 模板卡片 | 150px | 120px |
| 主题预览 | 100px | 80px |

### 4.2 默认尺寸

| 组件 | 默认宽度 | 默认高度 |
|------|---------|---------|
| 编辑器区域 | 40% | 100% |
| 预览区域 | 60% | 100% |
| 工具栏 | 100% | 48px |
| 状态栏 | 100% | 24px |

## 5. 交互设计

### 5.1 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + S` | 保存到 URL（更新分享链接） |
| `Ctrl/Cmd + E` | 导出面板 |
| `Ctrl/Cmd + Shift + C` | 复制为 PNG |
| `Ctrl/Cmd + Shift + S` | 复制分享链接 |
| `Ctrl/Cmd + B` | 切换引擎 |
| `Ctrl/Cmd + /` | 切换主题面板 |
| `F11` | 预览全屏 |

### 5.2 拖拽调整

- 编辑器/预览分割线：可左右拖拽调整比例
- 平板端分割线：可上下拖拽调整比例
- 预览区域：可拖拽平移图表

### 5.3 触摸操作（移动端）

| 操作 | 功能 |
|------|------|
| 双指捏合 | 缩放预览 |
| 单指拖拽 | 平移预览 |
| 双击 | 适应窗口 |
| 左滑 | 切换到下一个 Tab |
| 右滑 | 切换到上一个 Tab |

## 7. 颜色变量

```css
/* 编辑器 */
--editor-bg: #1e1e1e;           /* Monaco 暗色背景 */
--editor-line-highlight: #264f78; /* 当前行高亮 */
--editor-selection: #264f78;     /* 选中背景 */

/* 预览区 */
--preview-bg: #ffffff;          /* 预览背景 */
--preview-grid: #f0f0f0;        /* 网格线（可选） */

/* 错误 */
--error-line-bg: rgba(255, 0, 0, 0.1); /* 错误行背景 */
--error-underline: #f44336;     /* 错误波浪线 */
--error-panel-bg: #fff3f3;      /* 错误面板背景 */
--error-text: #c62828;          /* 错误文字 */
```
